This is a simple music player that currently supports mp3, wav, ogg

Shortcut Description:

W: Previous

S: Next

A: Volume up

D: Volume down

E: Stop/restart playback

Spacebar: Pause play/resume play

Z: Random Play

X: Select file to play

C: Add to queue

R: Empty queue

V: Change playback mode

Q: Quit
